# Stringo
Let's play with strings

## Installation
Using npm:
```bash
npm install --save stringo
```

## Using Stringo

In Node.js
```bash
const Stringo = require('stringo');
```

To run the tests:
```bash
npm test
```

**Note:** You need to install the latest version of [Jest](https://jestjs.io/) to run the tests
